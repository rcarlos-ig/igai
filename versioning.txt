Semantic versioning

First Number: Tracks major changes
Second Number: Tracks minor changes
Third Number: Tracks patches or mere bug fixes


First release                       New product       Start with 1.0.0                1.0.0
Backward compatible bug fixes	      Patch release     Increment the third digit       1.0.1
Backward compatible new features	  Minor release     Increment the middle digit
                                                      and reset last digit to zero	  1.1.0
Changes that break backward
compatibility                       Major release	    Increment the first digit and
                                                      reset middle and last digits
                                                      to zero                         2.0.0
